Troubleshooting

Fix common problems.

Start with the smallest check first: confirm the installed version, then test direct startup, then repair or reinstall.

Status and update repair

aerop2p status aerop2p update

If the command is missing, restart the terminal or add `~/.local/bin` to PATH.

Linux Executable does not start

chmod +x Aero-P2P-Chat-Linux.x86_64 ./Aero-P2P-Chat-Linux.x86_64

If the file starts manually but not from the launcher, reinstall the launcher metadata:

bash <(curl -sSL https://zorblock.github.io/AeroP2Pchat/install.sh) update

Black screen on Linux

Test with WebKit compositing disabled:

WEBKIT_DISABLE_COMPOSITING_MODE=1 aerop2p open

Also check whether the session is X11 or Wayland:

echo $XDG_SESSION_TYPE

Camera, mic, or screen share

  • Check desktop privacy permissions for microphone and camera.
  • Verify that another app is not locking the device.
  • On Wayland, screen sharing depends on the portal stack of the desktop environment.
  • Try the same action in an X11 session if Wayland screen capture fails.

Clean uninstall

aerop2p uninstall

The uninstaller asks whether user data and settings should be kept.
