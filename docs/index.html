Aero P2P Chat

A small desktop chat app for direct peer-to-peer conversations using Aero IDs. Available for Windows and Linux.

What it does

Simple desktop chat without accounts or a heavy service layer.

Direct chat

Share an Aero ID, connect, and start messaging.

Voice support

Use the built-in call controls when you need more than text.

Desktop updates

Check updates in-app or from the terminal with the CLI.

Linux install

Run the installer from GitHub Pages. It installs the AppImage and the `aerop2p` terminal command.

curl -fsSL "https://zorblock.github.io/AeroP2Pchat/install.sh" | sh
